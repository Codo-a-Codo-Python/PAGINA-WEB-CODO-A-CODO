Aprende HTML


¿Qué es HTML?

  • HTML viene de las siglas Hyper Text Markup Language
  • HTML es el lenguaje estándar de las páginas web
  • HTML describe la estructura de una página web
  • HTML consiste en una serie de elementos
  • Los elementos HTML le dicen al navegador como mostrar el contenido
  • Los elementos HTML etiquetan a las piezas de contenido tales como "Este es el encabezado", "Esto es un parrafo", "Esto es un link", etc.

Aprende CSS


¿Qué es CSS?

  • CSS viene de las siglas Cascading Style Sheets
  • CSS describe como los elementos HTML tienen que ser mostrados
  • CSS ahorra mucho trabajo. Puede controlar la disposición de múltiples paginas web al mismo tiempo
  • Las hojas de estilo externas se almacenan en archivos CSS

¿Querés saber más sobre CSS? Hace click aca


Aprende Java Script


¿Qué es JS?

  • JS puede cambiar el contenido HTML
  • JS puede cambiar los valores de los atributos HTML
  • JS puede cambiar los estilos de HTML (CSS)
  • JS puede esconder elementos HTML
